Display Property

In this section, we will learn Display (block, inline, inline-block, flex, grid)

  1. "display : block" :Full-width block-level element.
  2. "display : inline" :No new line, width fits content.
  3. "display : inline-block" :Like inline but allows setting width/height.
  4. "display : none" :Hides the element (no space taken).
  5. "display : flex" :Enables Flexbox layout.
  6. "display : grid" :Enables Grid layout.
  7. "display : table" :Behaves like an HTML table.
  8. "display : contents" :Only child elements are rendered.
  9. "display : inherit" :inherits parent's property
